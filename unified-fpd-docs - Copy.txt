# Unified Fancy Product Designer Documentation

This document combines the individual reference pages into a single file for easier consumption.

---

## Getting Started

First of all add the necessary JS and CSS files to your web page.

```html
<link rel="stylesheet" type="text/css" href="css/FancyProductDesigner.min.css" />
<script src="js/fabric-5.3.1.min.js" type="text/javascript"></script>
<script src="js/FancyProductDesigner.min.js" type="text/javascript"></script>
```

Create a container element for the product designer anywhere in your page.

```html
<div id="fpd-target" class="fpd-sidebar fpd-shadow-2"></div>
```

In the class attribute you can also define the initial layout (See demos folder).

Now initialize an instance of FancyProductDesigner on that container element.

```javascript
const appOptions = {};
const fpd = new FancyProductDesigner(
  document.getElementById('fpd-target'),
  appOptions
);
```

You can find all app options here.

### API

#### Methods

```javascript
fpd.getOrder(); //https://jsdoc.fancyproductdesigner.com/FancyProductDesigner#getOrder
//access current view instance
fpd.currentViewInstance.toDataURL() // export the current selected view as data URI
```

#### Events

```javascript
fpd.addEventListener('ready', () => {
  //wait for this event in order to make any api method call
})
```

The default options.

| `loadFirstProductInStage` | `Boolean` | `true` | Loads the first initial product into stage. |
| `unsavedProductAlert` | `Boolean` | `false` | Alert window will pop up if the user leaves the page without saving. |
| `hideDialogOnAdd` | `Boolean` | `true` | If the user adds something and off-canvas panel or dialog is opened, it will be closed. |
| `toolbarPlacement` | `String` | `'smart'` | Set the placement of the toolbar. |
| `fabricCanvasOptions` | `Object` | `{}` | An object containing options for the fabricjs canvas. |
| `namesNumbersDropdown` | `Array` | `[]` | Defines the values for the select element in the names & numbers module. |
| `namesNumbersEntryPrice` | `Number` | `0` | Sets price for any extra entry in the names & numbers module. |

### Methods

#### Options.merge(defaults, [merge]) ⇒ Object
Merges the default options with custom options.
- Returns: Object - The new options object.
- Parameters:
  - `defaults` (Object): The default object.
  - `[merge]` (Object): The merged object, that will be merged into the defaults.

#### Options.getParameterKeys() ⇒ Array
Returns all element parameter keys.
- Returns: Array - An array containing all element parameter keys.

## FancyProductDesigner

**Kind:** global class
**Extends:** EventTarget

### Constructor

#### new FancyProductDesigner(elem, [opts])
Creates a new FancyProductDesigner.
- Parameters:
  - `elem` (HTMLElement): The container for the Fancy Product Designer.
  - `[opts]` (Object): Options for configuration. Default `{}`.

### Static Properties

- FancyProductDesigner.additionalModules : Object
  You can register your own modules and add them in this static property.
  - Default: `{}`
  - Example: `{'my-module': ModuleClass}`

### Inner Properties
- `~container` (HTMLElement): The container for the Fancy Product Designer.
- `~mainOptions` (Object): The main options set for this Product Designer.
- `~currentViewInstance` (FancyProductDesignerView): The current selected view instance.
- `~currentViewIndex` (Number): The current selected view index.
- `~products` (Array): Array containing all products.
- `~designs` (Array): Array containing all designs.
- `~currentElement` (fabric.Object): The current selected element.
- `~currentPrice` (Number): The price considering the elements price in all views with order quantity.

### Methods

- setupDesigns(designs)
  Set up the designs with a JSON.
  - Parameters:
    - `designs` (Array): An array containg the categories with designs.

- addProduct(views, [category])
  Adds a new product to the product designer.
  - Parameters:
    - `views` (array): An array containing the views for a product.
    - `[category]` (string): If categories are used, you need to define the category title.

- loadProduct(views, [onlyReplaceInitialElements], [mergeMainOptions])
  Loads a new product to the product designer.

- addView(view)
  Adds a view to the current visible product.

- getIndex() ⇒ Number
  Gets the index of the view.

- selectView(index)
  Selects a view from the current visible views.

- getElements([viewIndex], [elementType], [deselectElement]) ⇒ Array
  Returns an array with fabricjs objects.

- getCustomElements([type], [viewIndex], [deselectElement]) ⇒ array
  Returns an array with all custom added elements.

- reset()
  Clears the product stage and resets everything.

- deselectElement()
  Deselects the selected element of the current showing view.

- addCustomImage(source, title, options, [viewIndex])
  Adds a new custom image to the product stage.

- getProduct([onlyEditableElements], [customizationRequired]) ⇒ array
  Returns the current showing product with all views and elements in the views.

- getProductDataURL(callback, [options], [viewRange])
  Creates all views in one data URL.

- print()
  Opens the current showing product in a Pop-up window and shows the print dialog.

- getOrder([options]) ⇒ object
  Returns an order object containing the product, usedFonts, and usedColors.

- calculatePrice([considerQuantity]) ⇒ Number
  Calculates the total price considering the elements price in all views and pricing rules.

### Events
- "ready": Gets fired as soon as product designer is ready.
- "productAdd": Gets fired when a product is added.
- "productSelect": Gets fired when a product is selected.
- "elementAdd": Gets fired when an element is added.
- "elementModify": Gets fired when an element is modified.
- "priceChange": Gets fired as soon as the price changes in a view.
- "viewSelect": Gets fired as soon as a view has been selected.

## FancyProductDesignerView

**Kind:** global class
**Extends:** EventTarget

### Constructor

#### new FancyProductDesignerView(container, [viewData], [callback], [fabricCanvasOptions])
Creates a new FancyProductDesignerView.
- Parameters:
  - `container` (HTMLElement): The container for the Fancy Product Designer View.
  - `[viewData]` (Object): The initial view data.
  - `[callback]` (function): Callback when view is created.
  - `[fabricCanvasOptions]` (Object): Options for fabricJS canvas.

### Inner Properties
- `~totalPrice` (Number): The total price for the view without max. price.
- `~truePrice` (Number): The total price for the view including max. price and correct formatting.
- `~additionalPrice` (Number): Additional price for the view.
- `~mask` (Object): The properties for the mask object.

### Methods

- init()
  This method needs to be called to initialize the generation.

- loadElements(elements, callback)
  Removes the current elements and loads a set of new elements into the view.
  - Parameters:
    - `elements` (Array): An array containing elements.
    - `callback` (function): A function that will be called when all elements have been added.

- toDataURL(callback, [options], [deselectElement])
  Creates a data URL of the view.
  - Parameters:
    - `callback` (function): A function that will be called when the data URL is created.
    - `[options]` (Object): Options object for data URL generation (`onlyExportable`, `backgroundColor`, etc.).
    - `[deselectElement]` (Boolean): Deselect current selected element.

- toSVG([options], [fontsToEmbed]) ⇒ String
  Returns the view as SVG.
  - Returns: String - A XML representing a SVG.

- toggleLock(toggle) ⇒ Boolean
  Toggles the lockment of view. If the view is locked, the price of the view will not be added to the total product price.

- changePrice(price, operator) ⇒ Number
  Changes the price by an operator, + or -.

### Events
- "priceChange": Gets fired as soon as the price has changed.

## FPDUtils

**Kind:** global class

A class with some static helper functions. You do not need to initiate the class, just call the methods directly, e.g. `FPDUtil.showModal();`

### Static Methods

- FPDUtils.arrayUnique(array) ⇒ Array
  Makes an unique array.
  - Returns: Array - Returns the edited array.
  - Parameters:
    - `array` (Array): The target array.

- FPDUtils.showModal(htmlContent, [fullscreen], [type], [container])
  Displays a modal dialog.
  - Parameters:
    - `htmlContent` (String): The html content for the modal.
    - `[fullscreen]` (Boolean): Displays the modal in full screen. Default: `false`.
    - `[type]` (String): Empty, 'prompt' or 'confirm'.
    - `[container]` (HTMLElement): The container for the modal. Default: `document.body`.

- FPDUtils.showMessage(text, [autoRemove])
  Displays a message in a snackbar (bottom-left).
  - Parameters:
    - `text` (String): The text for the message.
    - `[autoRemove]` (Boolean): Either to remove the message automatically or not. Default: `true`.

- FPDUtils.isUrl(s) ⇒ Boolean
  Checks if a string is an URL.
  - Returns: Boolean - Returns true if string is an URL.
  - Parameters:
    - `s` (String): The string.

- FPDUtils.pixelToUnit(pixel, [unit], [dpi]) ⇒ Boolean
  Converts a pixel value to any metric value considering the DPI.
  - Returns: Boolean - Returns true if string is an URL.
  - Parameters:
    - `pixel` (Number): The pixel value.
    - `[unit]` (String): Target metric - 'inch', 'mm', 'cm'. Default: `'inch'`.
    - `[dpi]` (Number): Target DPI. Default: `72`.

## fabric.Canvas

**Kind:** API Reference Extension

### Methods

- `addElement(elements, [callback])` ⇐ `fabric.Canvas`
  Adds a set of elements into the view.
  - Parameters:
    - `elements` (Array): An array containing elements.
    - `[callback]` (function): A function that will be called when all elements have beed added.

- `addElement(type, source, title, [parameters])` ⇐ `fabric.Canvas`
  Adds a new element to the view.
  - Parameters:
    - `type` (string): The type of an element you would like to add, 'image' or 'text'.
    - `source` (string): For image the URL to the image and for text elements the default text.
    - `title` (string): Only required for image elements.
    - `[parameters]` (object): An object with the parameters, you would like to apply on the element.

- `resetSize(discardActiveObject)` ⇐ `fabric.Canvas`
  Deselects the current selected element.

- `resetSize()` ⇐ `fabric.Canvas`
  Resets the canvas size considering the available space on the device.

- `setResZoom(value)` ⇐ `fabric.Canvas`
  Sets the zoom of the stage. 1 is equal to no zoom.

- `resetZoom()` ⇐ `fabric.Canvas`
  Resets the the zoom.

- `getElements()` ⇒ Array
  Returns an array with fabricjs objects.

- `getElementsJSON(title)` ⇒ Object
  Returns an fabric object by title.

- `getElementByTitle(title)` ⇒ Object
  Returns an fabric object by title.

- `getElementByID(id)` ⇒ Object
  Returns an fabric object by ID.

- `reset()` ⇐ `fabric.Canvas`
  Removes the canvas and resets all relevant view properties.

- `removeElement(element)` ⇐ `fabric.Canvas`
  Removes an element using the fabric object or the title of an element.
  - Parameters:
    - `element` (object | string): Needs to be a fabric object or the title of an element.

- `getElementByReplace()` ⇐ `fabric.Canvas`
  Gets an elment by replace property.

- `setElementOptions(parameters, [element])` ⇐ `fabric.Canvas`
  Sets the parameters for a specified element.
  - Parameters:
    - `parameters` (object): An object with the parameters that should be applied to the element.
    - `[element]` (fabric.Object | string): A fabric object or the title of an element.

- `duplicateElement(element)` ⇐ `fabric.Canvas`
  Duplicates an element in the canvas.

- `getUploadZone(title)` ⇒ fabric.Object
  Gets an upload zone by title.

- `setMask(maskOptions, [callback])` ⇐ `fabric.Canvas`
  Use a SVG image as mask for the whole view. The image needs to be a SVG file with only one path. The method toSVG() does not include the mask.
  - Parameters:
    - `maskOptions` (Object | Null): An object containing the URL to the svg. Optional: scaleX, scaleY, left and top.
    - `[callback]` (function): A function when mask is loaded and set. Returns the mask or null.

## Element (fabric.Object)

**Kind:** API Reference Extension

### Methods

- `getType([fabricType])` ⇒ String
  Gets the type of an element.
  - Returns: String - The element type - text or image.

- `isSVG()` ⇒ Boolean
  Checks if the element is a SVG.

- `isColorizable()` ⇒ Boolean
  Checks if the element is colorizable.

- `isBitmap()` ⇒ Boolean
  Checks if the element is a bitmap image.

- `hasColorSelection()` ⇒ Boolean
  Checks if the element has a color selection.

- `checkEditable(checkProps)` ⇒ Boolean
  Checks if the element can be edited by the user.

- `changeColor(colorData, [colorLinking])` ⇐ fabric.Canvas
  Changes the color of an element.
  - Parameters:
    - `colorData` (String | Array): Hex color value or any array of hex color values.
    - `[colorLinking]` (Boolean): If element is color linked, execute it.

- `setPattern(patternUrl)` ⇐ fabric.Canvas
  Sets the pattern for an element.

- `getZIndex()` ⇒ Number
  Gets the z-index of an element.

- `centerElement([hCenter], [vCenter])` ⇐ fabric.Object
  Centers an element horizontal or/and vertical.

- `getBoundingBoxCoords(element)` ⇒ Object | Boolean
  Returns the bounding box of an element.

- `getClippingObject()` ⇒ Object | fabric.Object
  Gets the object for the clipping. Could be an object with coordinates or a fabric.Object.

- `getElementJSON([addPropertiesToInclude], [propertyKeys])` ⇒ Object
  Gets the JSON representation of the element.
  - Returns: Object - A JSON representation of the element.

- `alignToPosition([pos])` ⇐ fabric.Canvas
  Aligns an element. Allowed values: left, right, top or bottom.

- `toggleUploadZone()` ⇐ fabric.Canvas
  Toggles the visibility of an upload zone. The upload zone is visible if it contains an element and hidden when empty.
